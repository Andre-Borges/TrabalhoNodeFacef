## Configuração


https://www.docker.com/get-started

1. Caso não tenha, instale o [Docker](https://www.docker.com/get-started)
  
   
2. Instale todas as dependências:

	```sh
	$ npm install
	```

3. Suba o banco de dados com o docker:

	```sh
	$ docker-compose up
	```
